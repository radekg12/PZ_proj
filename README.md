# PZ_proj
A basic Java application - Checkers Game, made as a university project.


## Used technology: 
* Java 8 : Swing
* MSSQL : mssql-jdbc
* REST WebService : fixer API, Json, okhttp, jackson-databind
* XML Parser
* Maven
* IntelliJ IDEA
  
  
More details [here](/Wymagania_projektu.md).
